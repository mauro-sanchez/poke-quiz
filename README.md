# PokeQuiz

Test your Poké-knowledge, test yourself.

To run this app, you'll need:

* Ruby 3.2.4

* Postgres ^12

And run this commands

* `bundle install`

* `rails db:setup`

* `rails s`

## Check users answers (API)

You can check user answers locally in `/api/v1/quizzes.json`
